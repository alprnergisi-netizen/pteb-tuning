Website Setup Guide
Your website is live and ready. Complete the 2 steps below to activate the buy buttons and contact form emails.
1
Shopify — Activate Buy Buttons
Connects your Shopify store so customers can purchase the Warport and Hoodie directly on the website.
A
Log into your Shopify admin at your-store.myshopify.com/admin
B
Go to Settings → Apps and sales channels → Develop apps → click Create an app. Name it "PTEB Website".
C
Click Configure Storefront API scopes. Enable: unauthenticated_read_product_listings and unauthenticated_write_checkouts. Save.
D
Click Install app. Under API credentials, copy your Storefront API access token. You'll need it below.
SHOPIFY_STORE_DOMAIN
Your store URL, e.g. ptebtuning.myshopify.com
SHOPIFY_STOREFRONT_ACCESS_TOKEN
The token you copied in step D above
SHOPIFY_WARPORT_HANDLE
Shopify product slug for Warport — default: pteb-warport
SHOPIFY_HOODIE_HANDLE
Shopify product slug for Hoodie — default: tuned-by-pteb-hoodie
⚠
Product handles: these must exactly match the URL slug of each product in Shopify. Find it in Shopify admin → Products → click the product → scroll to "Search engine listing" → the handle is the last part of the URL.
2
Resend — Activate Contact Form Emails
Makes the contact page and Warport enquiry form deliver emails to your inbox. Free plan is sufficient.
A
Go to resend.com and create a free account.
B
Go to API Keys → Create API key. Copy the key that starts with re_.
C
Go to Domains → Add domain. Enter ptebtuning.com. Resend will give you DNS records to add.
D
Add those DNS records in your domain registrar (wherever you registered ptebtuning.com). Wait ~5 min for verification.
RESEND_API_KEY
The key from step B above
RESEND_FROM_EMAIL
noreply@ptebtuning.com (once domain verified)
ADMIN_EMAIL
Email that receives all enquiries — already set to prestigeteamhelp@gmail.com
Free plan covers 3,000 emails/month — more than enough for enquiries. No credit card required.
3
Add Variables to Netlify & Redeploy
A
Log in at app.netlify.com → click your site (ptebtuning)
B
Go to Site configuration → Environment variables → Add a variable
C
Add each variable from Steps 1 and 2 above. Key in the left box, value in the right box.
D
Go to Deploys → Trigger deploy → Deploy site. Wait ~2 minutes. Done.
